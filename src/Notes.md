# Copilot Agents — Quick Reference

## @analyst — Pattern Recognition
```
@analyst <paste the problem statement here>
```
**Example:**
```
@analyst
A. Watermelon — Given w, check if it can be split into two even parts...
```

---

## @coder — Code Review & Hints
```
@coder <paste your code or describe what you're implementing>
```
**Examples:**
```
@coder Review my solution for CF 158A — here's my code: <paste code>
```
```
@coder I'm stuck implementing a segment tree for range sum queries
```

---

## @tester — Testing & Debugging
```
@tester <paste your code + the problem constraints>
```
**Examples:**
```
@tester My solution for Next Round gives WA on test 3. Here's my code: <paste code>
```
```
@tester What edge cases should I test for this greedy problem with N ≤ 50?
```

---

## Workflow for a New Problem
1. `@analyst` — paste the problem → understand the pattern & approach
2. Write your solution
3. `@coder` — paste your code → get review and bug hints
4. `@tester` — paste your code → get edge cases and debugging help
